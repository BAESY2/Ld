## OpenCode란?

터미널/IDE/데스크톱에서 코드를 작성해 주는 **오픈소스 AI 코딩 에이전트**입니다.
Claude Code의 오픈소스 대안으로 가장 많이 거론됩니다.

- GitHub 스타 16만+ , 기여자 900명+ , 월 750만+ 개발자 사용
- 라이선스: MIT (오픈소스)
- 터미널 우선(TUI) 인터페이스

## 핵심 특징

| 항목 | 내용 |
|------|------|
| **프로바이더 중립** | Anthropic, OpenAI, Google, OpenRouter 등 + 로컬 모델(Ollama, OpenAI 호환 엔드포인트) |
| **내장 에이전트** | `build`(전체 접근) / `plan`(읽기 전용) 두 가지 모드 |
| **LSP 통합** | 20개 이상 언어의 언어 서버 연동 |
| **MCP 지원** | Model Context Protocol 서버 연결 가능 |
| **프라이버시** | 코드/컨텍스트를 저장하지 않음 — 민감 환경에서 운용 가능 |

## Claude Code 대비 장점

1. **오픈소스(MIT)** — 자유로운 사용/수정/배포
2. **프로바이더 유연성** — 특정 벤더에 종속되지 않음
3. **로컬 모델 지원** — 오프라인/온프레미스 운용 가능
4. **프라이버시 우선** — 코드가 로컬을 벗어나지 않음

## Ladder AI에서 쓰는 이유

- 모델 프로바이더를 자유롭게 바꿔가며 비용/성능을 최적화할 수 있음
- 로컬 모델로 민감한 코드도 외부 유출 없이 처리 가능
- 오픈소스라 팀 워크플로에 맞게 커스터마이즈(에이전트/커맨드/플러그인) 가능

## 이 저장소에 적용한 설정

- `opencode.json` — 모델, 권한, 포맷터, LSP, 지침 파일 경로 설정
- `AGENTS.md` — 프로젝트 규칙/컨벤션 (에이전트가 자동으로 참조)
