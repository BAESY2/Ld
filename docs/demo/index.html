왜 아직 아무도 못 했나 — 안전과 결정론

LLM은 확률(환각). 공장은 1과 0의 결정론. 1mm·1접점 오작동이 수억 설비·인명 사고. 그래서 "자연어→즉시 기계"는 허상.

자연어 기획→ LLM(추출만) → 명세→ 결정론 합성 → ST/래더→ Z3·이중코일·컴파일 → 정형 검증→ 디지털트윈 → 가상 1:1 테스트→ 안전커널 → 실기 제어

초록 단계 = LLM 손 안 댐(결정론). 환각이 기계에 닿지 못하는 구조가 우리의 해자.
